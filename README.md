# Advanced Deep Learning for Computer Vision

Summary for TUM lecture "Advanced Deep Learning for Computer Vision (IN2390)". [Download PDF](https://github.com/weber-andreas/adl4cv)

## Content

The summary contains the following content:

- [x] Recap: Deep Learning
  - Siamese Networks for similarity learning
  - ConvNet visualization techniques (layer visualization, occlusion experiments)
  - Clustering visualization with t-SNE

- [x] 3D Data Representation
  - Volumetric Grids and 3D CNNs
  - Volumetric Hierarchies and Octrees

- [x] Neural Fields
  - MLP as a data structure
  - Implicit Neural Representations
  - Signed Distance Fields (SDF) and Occupancy Fields

- [x] Neural Radiance Fields (NeRF)
  - Core idea of continuous 5D function representation
  - Volumetric rendering and novel view synthesis
  - Probabilistic interpretation

- [x] Gaussian Splatting
  - Point-based splatting with surfels
  - 3D Gaussian Splatting with anisotropic ellipsoids
  - Spherical harmonics for color representation

- [x] Generative Neural Networks
  - Taxonomy of generative models
  - Autoencoders and Variational Autoencoders (VAEs)
  - Generative Adversarial Networks (GANs)

- [x] Conditional Generative Adversarial Networks (cGAN)
  - Conditioning generator and discriminator
  - Manipulating the latent manifold
  - Interactive GANs (iGANs) for image editing

- [x] Sequence Modelling
  - Text and images as sequences
  - Recurrent Neural Networks (RNNs) and long-term dependencies
  - Long Short-Term Memory (LSTMs)

- [x] Large Reconstruction Models (LRM)
  - Reconstruction vs. generation approaches
  - Feed-forward networks for 3D synthesis
  - PixelNeRF for image-conditioned generation

- [x] Diffusion Models
  - Denoising Diffusion Probabilistic Models (DDPMs)
  - Denoising Diffusion Implicit Models (DDIM) for accelerated sampling
  - Forward and reverse diffusion processes
  - Score-based generative modeling
